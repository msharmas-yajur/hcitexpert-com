<!-- wp:paragraph -->
<p><strong>1. Google AI Studio Prompt Gallery</strong></p>
<!-- /wp:paragraph -->

<!-- wp:paragraph -->
<p>This library is designed primarily for developers. It features prompts for a variety of use cases, such as marketing and content creation. It enables users to test prompts within the platform using Google's Gemini models.</p>
<!-- /wp:paragraph -->

<!-- wp:paragraph -->
<p>Google AI Studio <a href="https://www.youtube.com/redirect?event=video_description&amp;redir_token=QUFFLUhqa25FcWZsQkNUV1FwZUk5bWx5VXRRNWhOYXVrUXxBQ3Jtc0tuTW1KMVZobXJnXzJjNEg3VzR4dGlZYm1sNTFKWFlOT1VObzB6SGZTcFZIb2xQbnhZNEtBaEpVUV9oNjJ2MGJnWV9XXzF0cVRoeEhSMVZ6TVRzWlM5VEQ1NXRBa1lTVEFDQVMtUGQ0bDUyNEV3blExWQ&amp;q=https%3A%2F%2Faistudio.google.com%2Fapp%2Fprompts&amp;v=q1qav7x32yE" target="_blank" rel="noreferrer noopener">https://aistudio.google.com/app/prompts</a></p>
<!-- /wp:paragraph -->

<!-- wp:paragraph -->
<p><strong>2. OpenAI Examples</strong></p>
<!-- /wp:paragraph -->

<!-- wp:paragraph -->
<p>This is one of the earliest prompt libraries, released by OpenAI. It focuses on simpler, system-based prompts, useful for tasks like website creation.</p>
<!-- /wp:paragraph -->

<!-- wp:paragraph -->
<p>OpenAI Prompt Library <a href="https://platform.openai.com/docs/examples">https://platform.openai.com/docs/examples</a></p>
<!-- /wp:paragraph -->

<!-- wp:paragraph -->
<p><strong>3. Anthropic Claude Prompt Library</strong></p>
<!-- /wp:paragraph -->

<!-- wp:paragraph -->
<p>This resource, similar to OpenAI's library, is offered by Anthropic, the creators of the Claude LLM. It offers a collection of prompts, including system prompts for building custom GPTs. It's noted for having more comprehensive prompts compared to OpenAI's offering.</p>
<!-- /wp:paragraph -->

<!-- wp:paragraph -->
<p>Anthropic Prompt Library <a href="https://docs.anthropic.com/en/prompt-library/library">https://docs.anthropic.com/en/prompt-library/library</a></p>
<!-- /wp:paragraph -->

<!-- wp:paragraph -->
<p><strong>4. Lexica.art</strong></p>
<!-- /wp:paragraph -->

<!-- wp:paragraph -->
<p>This platform specializes in prompts for image generation, focusing on popular tools like Stable Diffusion, Midjourney, and DALL-E. </p>
<!-- /wp:paragraph -->

<!-- wp:paragraph -->
<p><strong>5. Role Prompting GitHub Repository</strong></p>
<!-- /wp:paragraph -->

<!-- wp:paragraph -->
<p>This GitHub repository provides a large collection of "role prompting" examples, where prompts guide the LLM to assume a specific role (e.g., travel guide, English translator). It's highlighted as a valuable resource, particularly for those new to prompting techniques.</p>
<!-- /wp:paragraph -->

<!-- wp:paragraph -->
<p><strong>6. Google Cloud Vertex AI Prompt Gallery</strong></p>
<!-- /wp:paragraph -->

<!-- wp:paragraph -->
<p>This comprehensive library is designed for developers, but is also useful for general users. It features detailed prompts that use hashtags and stars as delimiters to highlight key elements. The platform allows users to add examples, improving prompt clarity and enhancing outputs.</p>
<!-- /wp:paragraph -->

<!-- wp:paragraph -->
